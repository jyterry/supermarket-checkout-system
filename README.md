# Supermarket Checkout System

🚀 My first Python project on GitHub.

A simple supermarket checkout system built with Python and CSV.

## Features

* Load product information from a CSV file
* Search products by barcode
* Add products to a shopping cart
* Calculate the total price
* Complete checkout through the command line

## Project Structure

```text
supermarket-checkout-system/
│
├── main.py
├── products.csv
└── README.md
```

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/jyterry/supermarket-checkout-system.git
```

2. Navigate to the project directory:

```bash
cd supermarket-checkout-system
```

3. Run the program:

```bash
python main.py
```
